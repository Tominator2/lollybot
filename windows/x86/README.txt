This is the win32 version of node-hid and it has been tested on WinXP and Windows 7.  It was compiled with the win 32 version of Node.js version 0.9.1.  

It DOES NOT work the 64 bit version of Node.js or with versions 0.9.9 or 0.10.5.  I have not tested it with other versions.

Node.js:  http://nodejs.org/dist/v0.9.1 
node-hid: https://github.com/node-hid/node-hid
